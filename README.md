# CPP Code Editor

Minimal desktop C++ editor built with Electron, React, Monaco, and an integrated terminal.

## Features

- Monaco-based C++ editing
- Compile and run with `g++`
- Bottom pane for output or terminal
- Workspace explorer with file and folder CRUD
- Global settings via `settings.json`
- Snippet save/insert workflow with searchable picker

## Run

```bash
npm install
npm run dev
```

## Notes

- `g++` must be available in `PATH`
- some terminal options depend on what is installed on the host machine
